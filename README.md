## 🎯 Objetivo
Este repositório centraliza meus estudos teóricos e práticos realizados durante a graduação e cursos extracurriculares. O foco é documentar a evolução técnica em algoritmos, lógica e arquitetura de software.

---

## 📂 Organização do Repositório

### 1. ☕ Java & POO
* **Projetos:** Sistemas de modelagem, interfaces com Swing e integração com banco de dados.
* **Tópicos:** Encapsulamento, Herança, Polimorfismo e JDBC.

### 2. 🔵 Linguagem C & Estrutura de Dados
* **Conteúdo:** Implementação de Listas Encadeadas, Pilhas, Filas e Árvores.
* **Foco:** Manipulação de memória (`malloc`, `free`) e eficiência de algoritmos.

### 3. 🧮 Teoria & Matemática
* **Autômatos:** Implementação e estudo de autômatos finitos e gramáticas.
* **Métodos Numéricos:** Scripts para resolução de problemas matemáticos (Newton-Raphson, etc).

### 4. 🗄️ Banco de Dados
* **Tecnologias:** MySQL, PostgreSQL (**Neon**) e Modelagem ER.
* **Prática:** Scripts de criação de tabelas e queries complexas.
